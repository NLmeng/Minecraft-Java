## Requirements

## Running

### On Windows
1. Double-click **`run.bat`** _or_ open Command Prompt, go to the project root, and run:
   ```batch
   run.bat
   ```

### On macOS / Linux
1. Make **`run.sh`** executable:
   ```bash
   chmod +x run.sh
   ```
2. Then run:
   ```bash
   ./run.sh
   ```
